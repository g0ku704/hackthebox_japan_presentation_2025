Finding CVEs with Fuzzing in Any Language

Goktug Serez — 2025

Why Multi-Language Fuzzing?

  • Modern infra = Python, Go, Rust, Java
  • Memory safety != logic safety
  • Parsers fail across all languages
  • Fuzzing explores unexpected states
  • CVEs emerge from logical inconsistencies

Coverage-Guided Fuzzing (CGF)

C/C++ Vulnerable Function


int parse(const uint8_t* d, size_t s){
    if (s < 4) return 0;
    uint32_t len = *(uint32_t*)d; // unsafe
    char buf[64];
    memcpy(buf, d+4, len);        // overflow
    return buf[0];
}

C/C++ Fuzz Harness (libFuzzer)


extern "C" int LLVMFuzzerTestOneInput(const uint8_t* d, size_t s) {
    parse(d, s);
    return 0;
}

Run Command (C/C++)


clang++ -fsanitize=address,fuzzer parse.cc -o fuzz
./fuzz

Safe Languages Still Break

Python Attack Surface

  • yaml.load unsafe constructors
  • Regex catastrophic backtracking
  • urllib.parse domain confusion
  • Exception-based DoS chains

Python Vulnerable Code


import yaml
def load_conf(data):
    return yaml.load(data, Loader=yaml.FullLoader)

Atheris Harness


import atheris, vuln, sys

def TestOneInput(d):
    try:
        vuln.load_conf(d.decode("utf-8","ignore"))
    except Exception:
        pass

atheris.Setup(sys.argv, TestOneInput)
atheris.Fuzz()

Run Command (Python)


pip install atheris
python3 fuzz.py

(Atheris requires Python 3.8–3.10)

Go: Why It Crashes

Panics == crashes.

Go Vulnerable Code


func Parse(b []byte) int {
    var d map[string]interface{}
    yaml.Unmarshal(b, &d)
    return 100 / int(d["replicas"].(float64))
}

Go Fuzz Harness


func FuzzParse(f *testing.F) {
    f.Add([]byte("replicas: 1"))
    f.Fuzz(func(t *testing.T, data []byte) {
        Parse(data)
    })
}

Run Command (Go)


go test -run=^$ -fuzz=FuzzParse -fuzztime=30s

Rust: Logic Panics

Rust Vulnerable Code


pub fn parse_token(t: &str) -> &str {
    let parts: Vec<&str> = t.split('.').collect();
    parts[2] // panic if <3 parts
}

Rust cargo-fuzz Target


#![no_main]
use libfuzzer_sys::fuzz_target;

fuzz_target!(|data: &str| {
    let _ = mycrate::parse_token(data);
});

Run Command (Rust)


cargo install cargo-fuzz
cargo fuzz init
cargo fuzz run parse_token

Java: Serialization & Polymorphism

Java Vulnerable Code


public class Vuln {
    public static Object load(String data) throws Exception {
        var m = new ObjectMapper();
        return m.readValue(data, Object.class);
    }
}

Jazzer Harness


public class FuzzVuln {
    public static void fuzzerTestOneInput(FuzzedDataProvider d) {
        try { Vuln.load(d.consumeRemainingAsString()); }
        catch (Exception ignored) {}
    }
}

Run Command (Java)


jazzer --cp=./ --target_class=FuzzVuln
